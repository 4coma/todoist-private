Fonctionnalités principales

Les fonctionnalités disponibles dans l'application

Ajout vocal intelligent

Ajoutez des tâches par commande vocale avec extraction automatique des détails via OpenAI Whisper et GPT.

  • Transcription automatique
  • Extraction de dates et rappels
  • Liste de courses vocale
  • Multiples tâches en une phrase

Gestion de tâches avancée

Créez, organisez et suivez vos tâches avec une interface intuitive et moderne.

  • Sous-tâches infinies
  • Priorités personnalisables
  • Dates d'échéance avec rappels
  • Recherche rapide

Système de projets

Organisez vos tâches par projets avec des couleurs et icônes personnalisées.

  • Projets illimités
  • Couleurs personnalisables
  • Liste de courses spéciale
  • Vue d'ensemble globale

Suivi du temps

Timer intégré pour chaque tâche avec suivi du temps estimé vs temps réel.

  • Timer par tâche
  • Temps estimé vs réel
  • Pause/reprise automatique
  • Statistiques détaillées

Notifications intelligentes

Rappels programmables avec navigation directe vers vos tâches.

  • Rappels personnalisables
  • Notifications persistantes
  • Navigation directe
  • Gestion des permissions

Synchronisation cloud

Accédez à vos tâches sur plusieurs appareils avec synchronisation automatique.

  • Synchronisation Firebase
  • Multi-appareils
  • Sauvegarde automatique
  • Respect de la vie privée

Personnalisation complète

8 thèmes de couleurs, mode sombre/clair et animations fluides.

  • 8 thèmes disponibles
  • Mode sombre/clair
  • Animations fluides
  • Interface personnalisable

Export et import

Sauvegardez et restaurez vos données facilement en format JSON.

  • Export JSON complet
  • Import de sauvegarde
  • Sauvegarde locale sécurisée
  • Migration automatique
